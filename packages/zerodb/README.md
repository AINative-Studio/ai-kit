# @ainative/ai-kit-zerodb

AINative ZeroDB integration for vector storage and memory in AI Kit applications.

## Installation

```bash
npm install @ainative/ai-kit-zerodb
# or
pnpm add @ainative/ai-kit-zerodb
# or
yarn add @ainative/ai-kit-zerodb
```
