# @ainative/ai-kit-testing

Testing utilities and fixtures for AI Kit applications.

## Installation

```bash
npm install @ainative/ai-kit-testing vitest
# or
pnpm add @ainative/ai-kit-testing vitest
# or
yarn add @ainative/ai-kit-testing vitest
```

**Note:** This package requires `vitest` (v1.0.0 or higher) as a peer dependency. Make sure to install it in your project.
