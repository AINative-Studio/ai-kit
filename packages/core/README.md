# @ainative/ai-kit-core

Framework-agnostic core for AI Kit - streaming, agents, state management, and LLM primitives.

## Installation

```bash
npm install @ainative/ai-kit-core
# or
pnpm add @ainative/ai-kit-core
# or
yarn add @ainative/ai-kit-core
```
