# @ainative/ai-kit-auth

AINative authentication integration for AI Kit applications.

## Installation

```bash
npm install @ainative/ai-kit-auth
# or
pnpm add @ainative/ai-kit-auth
# or
yarn add @ainative/ai-kit-auth
```
