# @ainative/ai-kit-rlhf

AINative RLHF (Reinforcement Learning from Human Feedback) integration for AI Kit.

## Installation

```bash
npm install @ainative/ai-kit-rlhf
# or
pnpm add @ainative/ai-kit-rlhf
# or
yarn add @ainative/ai-kit-rlhf
```
